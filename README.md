# 🕯️ Memorial Vela

Projeto interativo de um memorial digital com vela animada, efeitos de luz, fumaça e sistema de homenagem.

---

## 🚀 Funcionalidades

- 🕯️ Vela interativa (acender/apagar)
- 🔥 Chama animada
- 💨 Fumaça ao apagar
- ✍️ Homenagem em texto
- 💾 Salvamento automático (localStorage)
- ⏱️ Timer ativo
- ✨ Partículas visuais
- 🌙 Modo escuro

---

## 💻 Tecnologias

- HTML5
- CSS3
- JavaScript puro
